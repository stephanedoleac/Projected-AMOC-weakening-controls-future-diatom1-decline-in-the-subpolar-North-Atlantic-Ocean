# Overview
This repository contains the codes used in our paper "Projected AMOC weakening controls future diatom decline in the subpolar North Atlantic Ocean".

- implementation_Brewin2017.py : Implement the method developped by Brewin et al. (2017) to reconstruct phytoplankton size-classe NPP
- implementation_Zhong2024.py : Implement the method developped by Zhong et al. (2024) to estimate surface NO3 concentrations

# Installation
Required modules :
- numpy (https://numpy.org/)
- xarray (https://docs.xarray.dev/en/stable/)
- netCDF4 (https://unidata.github.io/netcdf4-python/)

Tested versions :
- v3.9.7

You can install those packages in a few minutes either with ```pip3``` or with ```conda```.
